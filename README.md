# Deep Origins Overlays Revamped [BETA]
Newer version of Deep Origins Overlays ( https://github.com/devoneox/deep-origins-overlays/ ) currently in development.
The current Deep Origins Overlays resourcepack will be marked as legacy and the new one will be in development on GitHub. Helping a little is appreciated as recoding an entire pack may take some time. 

# FAQ

## Why making an entire pack ?
I have to make new textures, add compatibility with fusion, fix existing bugs and add new overlay features. Making a new pack, even though it takes more time, is kind of easier to do than updating an existing one, as it consists of many different files, and I can lose myself very easily.

## Will it be compatible with Fusion ?
**Yes**, of course.

## Why will the textures change ?
**Well, there are two main reasons :**
1. I think retexturing some of the overlays will make the pack much more consistent, as some textures look a lot more simple than other ones.
2. Most of the overlays were actually borrowed from @kiddkany 's  **Overlay's** resourcepack, and due to license changes, I have to retexture them.

## Is the current pack discontinued ?
**No.** The current pack will continue to be updated to newer versions of Minecraft until this pack is fully released, but no features will be added.
